# Arduino Audio Alarm Clock

DIY audio alarm clock based on Arduino-compatible hardware.

## Features

- Real Time Clock (DS3231)
- Audio playback via DFPlayer Mini
- Multiple alarm sounds
- Compact design
- Future custom PCB

## Project Status

Planning phase

## Hardware

### Selected Components

| Component | Module |
|------------|----------|
| RTC | DS3231 |
| Audio | DFPlayer Mini |
